This site was built by Kirsten Opstad as a project for CS50x, an online computer science course offered by Harvard University.

The front end is supported by the Bootstrap framework and basic programming is written in javaScript.

The intention of this site is to demonstrate the basic spoon theory metaphor, to be used as an instrument for spreading awareness.

Based on the Christine Miserandino's spoon theory.
